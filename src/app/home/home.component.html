MNIST Digit Prediction

The MNIST data set of handwritten digits is a subset of a larger set available from NIST. MNIST contains a training set of 60,000 samples and test set of 10,000 samples. The digits have been size-normailised and centered in a fixed-size image. This application employs the MNIST data set in order to train an artificial neural network in order to make prediction of images and digit drawing.
